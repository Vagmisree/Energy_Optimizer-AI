## How to Use

1. **Install & Run:**
   ```bash
   npm install
   npm run dev
   ```

2. **Navigate to Dashboard:**
   - Open http://localhost:3000
   - All features fully functional

3. **Try Key Flows:**
   - Go to /analyze to run predictions
   - Check /insights for AI analysis
   - Visit /gamification for achievements
   - Try voice commands in /assistant
