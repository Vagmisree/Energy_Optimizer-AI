## 🏗️ Architecture

### Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4 with custom theme
- **UI Components**: shadcn/ui (25+ components)
- **Charts**: Recharts with custom styling
- **Icons**: Lucide React
- **Theme**: Dark/Light mode with Oklch color space
- **Animations**: Custom CSS animations
- **State Management**: React hooks + Client components

### Component Structure
```
components/
├── premium-ai-features.tsx       # AI insights, personas, forecasts
├── ai-recommendations.tsx         # Smart recommendations, goals
├── ai-copilot.tsx               # Chat interface
├── notifications.tsx            # Alert system
├── premium-dashboard-metrics.tsx # Advanced visualizations
├── stat-card.tsx                # KPI cards
├── animated-counter.tsx         # Number animations
├── app-sidebar.tsx              # Navigation
├── charts/
│   ├── usage-chart.tsx
│   ├── bill-chart.tsx
│   └── appliance-chart.tsx
└── ui/                          # 25+ shadcn components
```

### Pages
- `/` - Dashboard (Main Hub)
- `/insights` - AI Intelligence Center
