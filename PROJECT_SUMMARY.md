## 💻 How to Deploy

### Quick Start
```bash
# Install dependencies
pnpm install

# Run development
pnpm dev

# Build production
pnpm build

# Deploy to Vercel
git push origin main
```

### Or Deploy Directly
```bash
# Deploy to Vercel
vercel deploy

# Or self-host
npm start
```
